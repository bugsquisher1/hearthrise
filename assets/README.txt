# Hearthrise Asset Split

These are cropped from the generated Hearthrise brand board.

Recommended name replacements:
- Crystal Caverns -> Glasshollow
- Ember Depths -> Cinderwake
- Ancient Grounds -> Rootgrave

Folders:
- logos/ = logo and emblem crops
- backgrounds/ = menu/banner/welcome art
- dungeons/ = dungeon cards and art-only thumbnails
- ui/ = UI strips and rough icon crops
- source/ = original full brand board

Note: The dungeon cards still contain the original generated text in the card versions. Use the `*_art_only.png` versions if you want to overlay the updated names in your game UI.
